# Exercises on Java Basics

A collection of foundational Java exercises completed as a rigorous self-study program to solidifying core programming mechanics. This repository serves as a practical sandbox to reinforce fundamental coding principles, software logic, and algorithmic thinking.

## Purpose & Context

These exercises were strategically completed as hands-on preparation and concurrent reinforcement for the **GTx CS1331: Introduction to Object-Oriented Programming with Java** professional certificate course. The goal is to build muscle memory for Java syntax and basic problem-solving before tackling advanced OOP concepts.

The problem sets implemented in this repository are sourced from the widely acclaimed academic notes by **Dr. Chua Hock Chuan** from **Nanyang Technological University (NTU), Singapore**. 

* **Source Material**: [Java Programming Tutorial: Exercises on Java Basics](https://www3.ntu.edu.sg/home/ehchua/programming/java/J2a_BasicsExercises.html)
